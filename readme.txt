This plugin allow you to display pop-up window with form for clients.You can choose posts where pop-up will be shown.
After submit, data from form will be saved on WordPress Database (wp_leads_catcher table), this informations will be
also displayed in plugin administrator's panel. Besides of sending data to database, you and your client will recive
message about this. All features like changing logo, email or adding tags which will trigger pop-up you can
configure/add in administrator's panel.

!IMPORTANT!
This plugin allows you to change only WORDPRESS OWNER'S EMAIL, if you want to configure pop-up email's form destination you need to configure your SMTP server first.
